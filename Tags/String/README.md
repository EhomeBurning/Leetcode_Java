# String 
## 4. String Reversal
### Method 1:   
Iteration(two pointers different direction).   
Time: O(n);   
Space: O(1);   

### Method 2: 
Recursive:    
1. base case: 0 or 1 char remaining.  i >= j.   
2. recursive rule: reverse(i + 1, j - 1).
